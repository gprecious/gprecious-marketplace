# 바이럴 메카닉스

콘텐츠가 바이럴되는 원리와 적용 전략.

## 바이럴의 과학

### 바이럴 공식
```
바이럴 = 콘텐츠 가치 × 공유 동기 × 공유 용이성
```

### 공유 심리학
1. **사회적 화폐**: 공유하면 나도 똑똑해 보임
2. **트리거**: 특정 상황에서 떠오름
3. **감정**: 강한 감정을 유발
4. **공개성**: 다른 사람도 공유하는 걸 봄
5. **실용적 가치**: 유용한 정보
6. **스토리**: 이야기로 포장됨

## Shorts 바이럴 요소

### 1. 첫 인상 (0.5초)
```
스크롤 멈추게 하는 요소:
- 시각적 충격
- 예상치 못한 것
- 강렬한 색상/움직임
- 얼굴/눈 (본능적 주목)
```

### 2. 호기심 갭
```
"이게 뭐지?" 반응 유발
정보를 약간만 보여주고 나머지는 숨김
알고 싶어서 계속 시청
```

### 3. 감정 유발
| 감정 | 바이럴 효과 | 예시 |
|------|------------|------|
| 놀라움 | 매우 높음 | "와 이게 가능해?" |
| 경외 | 높음 | "세상에 이런 일이" |
| 재미 | 높음 | "ㅋㅋㅋ 이거 봐봐" |
| 분노 | 높음 | "이건 좀 아니지" |
| 불안 | 중간 | "나도 조심해야겠다" |
| 슬픔 | 중간 | 감동 스토리 |

### 4. 공유 가치
```
공유하고 싶은 콘텐츠:
- 남들이 모르는 정보 (사회적 화폐)
- 반응이 궁금한 정보 (토론 유발)
- 유용한 정보 (실용적 가치)
- 재미있는 정보 (즐거움 공유)
```

## 바이럴 트리거

### 1. 반직관적 정보
```
상식: "운동하면 살이 빠진다"
반직관: "운동해도 살이 안 빠지는 이유"
→ 놀라움 + 호기심
```

### 2. 숨겨진 진실
```
"아무도 안 알려주는"
"숨겨진 비밀"
"진짜 이유"
→ FOMO + 독점 정보 욕구
```

### 3. 논쟁 유발
```
"A vs B, 당신의 선택은?"
"이거 맞다고 생각하세요?"
→ 댓글 + 공유 증가
```

### 4. 실용적 팁
```
"이 방법 하나로"
"이것만 알면"
"5분만 투자하면"
→ 저장 + 공유
```

### 5. 감정적 스토리
```
개인적 경험
극복 스토리
감동적 결말
→ 공감 + 공유
```

## 바이럴 유형별 전략

### 정보형 바이럴
```
"이거 몰랐죠?"
→ 놀라운 사실 + 쉬운 설명
→ "이거 봐봐" 반응
```

### 엔터테인먼트 바이럴
```
"ㅋㅋㅋ 이건 웃기다"
→ 재미 + 예상치 못한 전개
→ 친구에게 공유
```

### 감정형 바이럴
```
"와 감동이다"
→ 스토리 + 감정적 결말
→ 공감 공유
```

### 논쟁형 바이럴
```
"이건 좀 아닌 것 같은데"
→ 논쟁 유발 + 의견 요청
→ 댓글 폭발

⚠️ 주의: 부정적 논쟁은 채널에 악영향
```

## 바이럴 실패 원인

### 1. 너무 뻔함
```
예측 가능한 전개
이미 본 것 같은 콘텐츠
```

### 2. 공유 동기 부재
```
재미없음
유용하지 않음
감정 유발 실패
```

### 3. 첫 인상 실패
```
스크롤 멈추지 않음
훅 약함
```

### 4. 낚시성
```
기대와 다른 내용
신뢰 상실
```
