# Content Strategy Guide

연령대/국가별 YouTube Shorts 컨텐츠 전략 가이드.

> **참조**: 설정값은 `trending-config.json`에서 관리됩니다.

---

## 연령별 전략

### Young (10-20대)

**핵심 키워드**: 빠름, 트렌드, 밈, 진정성

| 항목 | 설정 |
|------|------|
| **1순위 컨텐츠** | 코미디/밈 (67%), 트렌드/챌린지, 게이밍 |
| **감정 트리거** | FOMO, 사회적 화폐, 서프라이즈, 유머 |
| **훅 타임** | 1.5초 이내 |
| **페이스** | 빠름 (컷 전환 빈번) |
| **톤** | 에너제틱, 트렌디, 진정성, 공감 |

**DO**:
- 첫 1.5초에 강렬한 훅
- 트렌드 사운드/밈 적극 활용
- "That's so me" 공감 포인트
- 플랫폼 네이티브 느낌 (광고 티 X)
- 슬랭/신조어 OK

**DON'T**:
- 느린 전개
- 설교/훈계 톤
- 과도한 폴리시 (오히려 거부감)
- 구식 표현/트렌드

---

### Middle (30-50대)

**핵심 키워드**: 실용, 효율, 신뢰, 성취

| 항목 | 설정 |
|------|------|
| **1순위 컨텐츠** | How-to/라이프스타일, 재테크, 건강/피트니스 |
| **감정 트리거** | 성취감, 실용적 가치, 노스탤지어, 문제 해결 |
| **훅 타임** | 3초 이내 |
| **페이스** | 중간 (정보 전달 중심) |
| **톤** | 프로페셔널, 실용적, 신뢰감, 효율적 |

**DO**:
- 명확한 가치 제안 ("이렇게 하면 ~됩니다")
- 문제-해결 구조
- 데이터/팩트 기반 신뢰성
- 시간 존중 (핵심만 빠르게)
- 달성 가능한 목표 제시

**DON'T**:
- 시간 낭비하는 느낌
- 젊은 세대 슬랭/밈 억지 사용
- 근거 없는 주장
- 과장된 약속

---

### Senior (60-70대)

**핵심 키워드**: 따뜻함, 명확함, 향수, 편안함

| 항목 | 설정 |
|------|------|
| **1순위 컨텐츠** | DIY/취미, 뉴스/시사, 음악, 건강 |
| **감정 트리거** | 노스탤지어, 따뜻함, 명확함, 편안함 |
| **훅 타임** | 5초 이내 |
| **페이스** | 느림 (천천히 명확하게) |
| **톤** | 따뜻함, 명확함, 부드러움, 신뢰감 |

**DO**:
- 천천히 명확한 나레이션
- 큰 글씨, 선명한 비주얼
- 향수 자극 (과거 추억)
- 가족/커뮤니티 테마
- 실용적 정보 (건강, 취미)

**DON'T**:
- 빠른 전개/컷 전환
- 복잡한 기술 용어
- 트렌드 추종
- 혼란스러운 구성

## 국가별 전략

### 한국 (ko)

| 항목 | 내용 |
|------|------|
| **1순위** | 유머/개그 (48%), 엔터테인먼트, 펫, 음식 |
| **유머 스타일** | 집단적, 귀여움 (애교), 드라마틱 |
| **특징** | K-Pop 영향, 먹방 인기, 연예인 일상 |
| **피해야 할 것** | 공격적 유머, 논쟁적 주제 |

**전략**: 귀여움 + 음식 + 드라마틱 전개

## 유니버설 성공 공식

### 첫 3초의 법칙

```
0-1초: 시각적 훅 (놀라운 이미지/움직임)
1-2초: 질문 또는 대담한 주장
2-3초: 왜 계속 봐야 하는지 암시
```

### 감정 트리거 우선순위

| 순위 | 트리거 | 효과 |
|------|--------|------|
| 1 | **경이로움 (Awe)** | 공유율 최고 |
| 2 | **서프라이즈** | 주목도 최고 |
| 3 | **유머** | 참여율 최고 |
| 4 | **기쁨** | 긍정적 연상 |
| 5 | **노스탤지어** | 시니어 효과적 |

### 피해야 할 감정

- **슬픔** (저각성, 공유 감소)
- **무관심** (아무 반응 없음)
- **혼란** (이탈 유발)

## API 활용 가이드

### YouTube Data API

```
GET /youtube/v3/videos?
  chart=mostPopular&
  regionCode={region}&
  videoCategoryId={category}&
  maxResults=50
```

- **비용**: 1 quota unit/call
- **일일 한도**: 10,000 units
- **권장**: 200 calls/day 이하

### Google Trends (pytrends)

```python
from pytrends.request import TrendReq
pytrends = TrendReq()
pytrends.build_payload(['keyword'], geo='KR')
trending = pytrends.trending_searches(pn='south_korea')
```

- **비용**: 무료 (비공식)
- **권장**: 100 calls/day 이하
- **캐싱**: 24시간 권장

### NewsAPI

```
GET https://newsapi.org/v2/top-headlines?
  country={country}&
  category={category}
```

- **비용**: 무료 (100/day)
- **용도**: 뉴스 기반 주제 발굴
