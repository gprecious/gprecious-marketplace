# 도파민 트리거 패턴

뇌과학 기반 콘텐츠 최적화 패턴.

## 도파민 시스템 이해

### 기본 공식
```
도파민 방출 = 실제 보상 - 예측 보상

예측보다 좋으면 → 도파민 ↑ (기쁨)
예측과 같으면 → 도파민 유지
예측보다 나쁘면 → 도파민 ↓ (실망)
```

### 핵심 인사이트
> "도파민은 보상 자체가 아니라 **예측 오류**에 반응한다"

## 핵심 패턴

### 1. Variable Reward (가변 보상)
```
❌ 고정 보상
"3가지 방법을 알려드릴게요"
→ 뇌가 이미 예측 → 도파민 감소

✅ 가변 보상
"몇 가지가 있는데요... 마지막이 제일 충격이에요"
→ 불확실성 → 도파민 유지
```

**적용 예시:**
- "첫 번째는... 근데 더 놀라운 건..."
- "이것만 있는 줄 알았는데..."
- "사실 진짜 이유는 따로 있어요"

### 2. Information Gap (정보 공백)
```
호기심 = 알고 싶은 것 - 알고 있는 것

공백이 너무 작으면 → 관심 없음
공백이 너무 크면 → 포기
적절한 공백 → 최대 호기심
```

**적용 예시:**
- "이 사진에서 이상한 점을 찾으셨나요?"
- "왜 그럴까요?"
- "정답은... (pause) ...바로 여기에 있습니다"

### 3. Prediction Loop (예측 루프)
```
예측 → 검증 대기 → 결과 확인 → 새로운 예측

이 루프가 반복될 때 몰입 유지
```

**적용 예시:**
- "이게 가능할까요?" → 실험 → 결과 → "더 놀라운 건..."
- "A일까요? B일까요?" → 정답 → "근데 왜 그럴까요?"

### 4. Anticipation (기대감)
```
기대하는 동안 도파민이 가장 높음
결과를 받는 순간보다 기다리는 동안이 더 행복

→ 정보를 바로 주지 말고 기대하게 만들기
```

**적용 예시:**
- "3초 후에 정답 공개합니다"
- "영상 끝까지 보면 알게 돼요"
- "이건 나중에 다시 말씀드릴게요"

### 5. Social Proof (사회적 증거)
```
"대부분의 사람들이 모르는..."
→ FOMO (놓치는 것에 대한 두려움) 유발
→ 도파민 시스템 활성화
```

**적용 예시:**
- "상위 1%만 아는"
- "전문가들도 놀란"
- "아무도 안 알려주는"

### 6. Novelty (새로움)
```
새로운 정보 = 도파민 스파이크
익숙한 정보 = 도파민 감소

15초마다 새로운 정보/자극 필요
```

**적용 예시:**
- 새로운 사실 공개
- 반전
- 예상치 못한 전개

## Shorts 적용 전략

### 타임라인 최적화
```
0-0.5초: 즉시 주목 (충격/질문)
0.5-3초: 호기심 유발 (정보 공백)
3-15초: 첫 번째 정보 공개
15-30초: Variable Reward (추가 정보)
30-45초: 반전/클라이맥스
45-60초: 미해결 미스터리 또는 CTA
```

### 도파민 트리거 밀도
- 10초당 최소 1개 트리거
- 15초 이상 트리거 없으면 위험
- 연속 트리거로 몰입 유지

### 연령대별 조정
| 연령 | 트리거 빈도 | 선호 트리거 |
|------|------------|------------|
| 10대 | 5초마다 | 시각적, 충격 |
| 20대 | 8초마다 | Variable Reward |
| 30대 | 10초마다 | 정보 공백 |
| 40대+ | 12초마다 | 기대감, 공감 |

## 피해야 할 패턴

### 1. 예측 가능한 전개
```
❌ "첫째는 A, 둘째는 B, 셋째는 C"
→ 뇌가 패턴 인식 → 도파민 감소
```

### 2. 즉시 정보 공개
```
❌ "이건 3km입니다"
✅ "얼마나 길까요? ... 무려 3km예요"
```

### 3. 긴 설명
```
❌ "왜냐하면 A이고 B이기 때문에 C가 되어서..."
✅ "이유는요. A. 끝."
```
