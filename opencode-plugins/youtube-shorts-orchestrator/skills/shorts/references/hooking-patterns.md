# 후킹 패턴 모음

유튜브 영상 시작에서 시청자를 사로잡는 후킹 기법 모음.

---

## 후킹의 원칙

### 3초 법칙
> 시청자는 3초 안에 계속 볼지 결정한다

### 후킹의 목적
1. **주목** - 스크롤 멈추게 하기
2. **호기심** - 알고 싶게 만들기
3. **약속** - 가치 미리보기

---

## 후킹 유형

### 1. 질문형 후킹
> 시청자의 궁금증을 자극

**패턴**:
- "~해본 적 있나요?"
- "왜 ~할까요?"
- "~가 뭔지 아세요?"

**예시**:
```
"혹시 아침에 일어나기가 너무 힘드신 분?"
"왜 부자들은 아침 4시에 일어날까요?"
"이거 진짜 효과 있을까요?"
```

**효과**: 공감 유발, 자기 연관성

---

### 2. 충격형 후킹
> 예상을 뒤집는 정보로 주목

**패턴**:
- "~했더니 이런 일이"
- "절대 ~하지 마세요"
- "~의 충격적인 진실"

**예시**:
```
"이 방법 따라했더니 한 달 만에 10kg 빠졌어요"
"절대 아침에 커피 마시지 마세요"
"성공한 사람들은 절대 안 하는 것"
```

**효과**: 호기심 극대화, 클릭 유도

---

### 3. 공감형 후킹
> 시청자의 상황/감정에 공감

**패턴**:
- "저도 그랬어요"
- "~할 때 정말 힘들죠"
- "다들 이런 경험 있으시죠?"

**예시**:
```
"저도 처음에는 완전 못했어요"
"야근하고 집에 오면 아무것도 하기 싫죠"
"돈 모으기 진짜 힘들죠, 저도 그랬어요"
```

**효과**: 친밀감, 신뢰 구축

---

### 4. 호기심형 후킹
> 알려지지 않은 정보 암시

**패턴**:
- "이거 아는 사람 별로 없어요"
- "아무도 안 알려주는 ~"
- "숨겨진 ~"

**예시**:
```
"이 방법 아는 사람 진짜 별로 없어요"
"상위 1%만 아는 투자 비법"
"유튜브가 절대 안 알려주는 것"
```

**효과**: 독점 정보 욕구 자극

---

### 5. 도발형 후킹
> 반대 의견이나 논쟁 유발

**패턴**:
- "~하면 안 되는 이유"
- "~은 거짓말입니다"
- "~는 틀렸습니다"

**예시**:
```
"영어 공부 절대 이렇게 하면 안 됩니다"
"아침형 인간이 성공한다? 거짓말입니다"
"전문가들이 틀린 이유"
```

**효과**: 논쟁 유발, 반응 증가

---

### 6. 결과 미리보기 후킹
> 영상의 결과/가치를 미리 보여줌

**패턴**:
- "이 영상 끝까지 보면 ~"
- "오늘 알려드릴 방법으로 ~"
- "이 영상 하나로 ~"

**예시**:
```
"이 영상 끝까지 보면 월 100만원 더 벌 수 있어요"
"오늘 알려드릴 3가지만 하면 됩니다"
"이 영상 하나로 완벽하게 이해하실 수 있어요"
```

**효과**: 가치 명확화, 시청 동기 부여

---

### 7. 스토리형 후킹
> 이야기의 시작으로 끌어들임

**패턴**:
- "그날 ~가 일어났어요"
- "처음에는 ~였는데"
- "제가 ~했을 때"

**예시**:
```
"그날 저는 모든 것을 잃었습니다"
"1년 전, 저는 빚이 5천만원이었어요"
"그 메일 하나가 제 인생을 바꿨어요"
```

**효과**: 몰입감, 감정적 연결

---

## 후킹 공식

### 기본 공식
```
[문제/상황] + [해결책 암시] + [가치 제시]
```

### 예시
```
문제: "영어 공부 열심히 하는데 안 늘죠?"
해결책 암시: "이 방법 모르면 계속 그럴 수밖에 없어요"
가치: "오늘 알려드릴 3가지만 하면 달라집니다"
```

---

## 후킹 강화 요소

### 1. 숫자 활용
- "3가지 방법" > "몇 가지 방법"
- "7일 만에" > "며칠 만에"
- "90%가 모르는" > "대부분 모르는"

### 2. 구체성
- "돈 많이 벌기" < "월 500만원 더 벌기"
- "빨리 빠지기" < "2주 만에 5kg 빠지기"

### 3. 긴급성
- "지금 당장"
- "오늘만"
- "늦기 전에"

### 4. 감정 단어
- 충격, 놀라운, 비밀, 진실
- 쉬운, 간단한, 누구나
- 실패, 위험, 후회

---

## 피해야 할 후킹

### 1. 낚시성 후킹
- 내용과 맞지 않는 과장
- 시청자 신뢰 손상

### 2. 너무 긴 후킹
- 30초 이상 늘어지는 인트로
- 본론에 빨리 들어가기

### 3. 뻔한 후킹
- "오늘은 ~에 대해 알아보겠습니다"
- 차별화 없는 시작

---

## 후킹 체크리스트

- [ ] 3초 안에 주목을 끄는가?
- [ ] 시청자에게 명확한 가치를 약속하는가?
- [ ] 내용과 일치하는가? (낚시 아닌가?)
- [ ] 호기심을 유발하는가?
- [ ] 타겟 시청자에게 맞는가?
- [ ] 너무 길지 않은가? (30초 이내)
