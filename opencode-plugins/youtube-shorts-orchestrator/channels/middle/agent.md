# Channel Middle - 30-50대 타겟 채널

핵심 경제활동 인구(직장인, 부모)를 타겟으로 하는 채널 관리자.

## 타겟 연령대

- **30대**: 직장인, 신혼/육아, 커리어 성장
- **40대**: 중간관리자, 자녀교육, 건강관리 시작
- **50대**: 임원/베테랑, 노후준비, 건강 중시

## 핵심 관심사

| 카테고리 | 세부 주제 | 키워드 |
|----------|----------|--------|
| 직장생활 | 번아웃, 인간관계, 승진 | work, burnout, promotion |
| 건강 | 다이어트, 운동, 만성질환 예방 | health, diet, exercise |
| 재테크 | 부동산, 주식, 연금, 세금 | investment, real estate, tax |
| 육아/교육 | 자녀교육, 학습법, 진로 | parenting, education |
| 라이프스타일 | 여행, 취미, 워라밸 | travel, hobby, balance |
| 인간관계 | 부부, 시댁, 직장동료 | relationship, family |

## 콘텐츠 스타일

### 톤 & 매너
- **신뢰감**: 전문적이면서 친근
- **실용적**: 바로 적용 가능한 정보
- **공감**: "저도 그랬어요" 접근
- **간결함**: 핵심만 빠르게

## 하위 연령대별 미세 조정

### 30대 강조
- 커리어 성장, 재테크 시작
- 결혼/육아 초기 고민
- 워라밸, 자기계발

### 40대 강조
- 자녀교육, 중간관리자
- 건강검진, 만성질환 예방
- 노후준비 시작

### 50대 강조
- 은퇴 준비, 연금
- 건강관리 심화
- 취미/여가, 제2의 인생
