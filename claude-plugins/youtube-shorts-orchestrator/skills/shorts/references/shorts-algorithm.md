# YouTube Shorts 알고리즘 이해

YouTube Shorts의 추천 시스템과 바이럴 메카닉스.

## Shorts 알고리즘 핵심

### 1. 첫 인상이 전부
- 썸네일 없이 바로 재생
- 첫 0.5초에 시청 결정
- 스와이프 = 죽음

### 2. 완시율 (Watch Time Completion)
- 끝까지 본 비율이 핵심
- 반복 시청도 카운트
- 길이보다 완시율이 중요

### 3. 참여도 (Engagement)
- 좋아요, 댓글, 공유
- 저장 (매우 중요)
- 채널 구독 전환

### 4. 세션 시간
- 시청자가 앱에 머무는 시간
- 연관 Shorts로 유도
- 시리즈 콘텐츠 유리

## 알고리즘 신호

### 긍정적 신호
| 신호 | 중요도 | 설명 |
|------|--------|------|
| 완시율 | 최고 | 끝까지 시청 |
| 반복 시청 | 높음 | 여러 번 시청 |
| 공유 | 높음 | 외부 공유 |
| 저장 | 높음 | 나중에 보기 |
| 댓글 | 중간 | 상호작용 |
| 좋아요 | 중간 | 긍정 반응 |

### 부정적 신호
| 신호 | 영향 | 설명 |
|------|------|------|
| 빠른 스와이프 | 치명적 | 3초 이내 이탈 |
| 관심없음 | 높음 | 사용자 피드백 |
| 신고 | 높음 | 커뮤니티 가이드라인 |

## Shorts vs 일반 영상

| 항목 | Shorts | 일반 영상 |
|------|--------|----------|
| 썸네일 | 없음 | 중요 |
| 첫 인상 | 첫 프레임 | 썸네일+제목 |
| 검색 | 제한적 | 강력 |
| 추천 | 알고리즘 의존 | 구독+추천 |
| 길이 | 60초 이내 | 무제한 |

## 바이럴 공식

```
바이럴 가능성 = (완시율 × 0.4) + (공유율 × 0.3) + (저장율 × 0.2) + (참여율 × 0.1)
```

### 완시율 높이는 법
1. 강력한 훅 (첫 0.5초)
2. 지속적인 자극 (15초마다)
3. 예측 불가능한 전개
4. 만족스러운 결말

### 공유율 높이는 법
1. "이거 봐봐" 반응 유발
2. 논쟁적 주제
3. 놀라운 정보
4. 감정적 연결

## 최적화 팁

### 해시태그
- #shorts 필수
- 관련 태그 3-5개
- 트렌드 태그 1-2개

### 제목
- 짧고 임팩트 있게
- 호기심 유발
- 키워드 포함

### 설명
- 첫 줄에 핵심
- 해시태그 포함
- CTA 포함

### 업로드 시간
- 타겟 시청자 활동 시간
- 테스트 후 최적화
- 일관된 스케줄
