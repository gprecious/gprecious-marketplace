# 연령대별 인사이트

10대부터 70대까지 연령별 특성과 콘텐츠 전략.

## 연령대별 특성

### 10대 (13-19세)

**미디어 소비 특성:**
- 극도로 짧은 집중 시간 (3-5초)
- 트렌드에 민감
- 밈 언어 능숙
- 동시 멀티태스킹

**관심사:**
- 틱톡/인스타 트렌드
- 게임, 스트리머
- K-POP, 연예인
- 학교생활, 친구

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 매우 빠르게 |
| 톤 | 자극적, 슬랭 OK |
| 훅 | 시각적 충격 |
| 길이 | 15-30초 권장 |

**예시 문구:**
- "ㄹㅇ 미쳤다"
- "이거 실화냐"
- "찐으로"

---

### 20대 (20-29세)

**미디어 소비 특성:**
- 짧은 집중 시간 (5-8초)
- 정보 소비 욕구 높음
- SNS 활발
- 출퇴근 시간 활용

**관심사:**
- 자기계발, 생산성
- 재테크, 투자
- 취업, 커리어
- 연애, 관계

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 빠르게 |
| 톤 | 친근, 현실적 |
| 훅 | 공감 + 정보 |
| 길이 | 30-45초 권장 |

**예시 문구:**
- "이거 진짜야"
- "나만 몰랐어?"
- "솔직히 말해서"

---

### 30대 (30-39세)

**미디어 소비 특성:**
- 보통 집중 시간 (8-10초)
- 실용 정보 선호
- 시간 효율 중시
- 육아 중 틈틈이

**관심사:**
- 직장생활, 연봉
- 재테크, 부동산
- 육아, 가정
- 건강 관리

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 적당히 빠르게 |
| 톤 | 실용적, 공감적 |
| 훅 | 문제 해결 제시 |
| 길이 | 35-50초 권장 |

**예시 문구:**
- "이건 알아두세요"
- "저도 겪어봤는데요"
- "실제로 효과가 있었어요"

---

### 40대 (40-49세)

**미디어 소비 특성:**
- 긴 집중 시간 (10-12초)
- 신뢰 기반 소비
- 검증된 정보 선호
- 아침/저녁 시간 활용

**관심사:**
- 자녀교육, 입시
- 건강, 검진
- 노후 준비
- 여행, 취미

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 차분하게 |
| 톤 | 신뢰감, 전문적 |
| 훅 | 근거 기반 호기심 |
| 길이 | 40-55초 권장 |

**예시 문구:**
- "연구에 따르면"
- "알아두시면 좋습니다"
- "실제로 많은 분들이"

---

### 50대 (50-59세)

**미디어 소비 특성:**
- 긴 집중 시간 (12-15초)
- 건강 정보 민감
- 가족 중심 사고
- 여유로운 시청

**관심사:**
- 건강, 갱년기
- 노후 준비
- 자녀, 손주
- 취미, 여행

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 여유있게 |
| 톤 | 따뜻하고 안정적 |
| 훅 | 공감 + 해결책 |
| 길이 | 40-55초 권장 |

**예시 문구:**
- "많이들 궁금해하시죠"
- "건강하게 지내시길"
- "이런 방법도 있어요"

---

### 60대 (60-69세)

**미디어 소비 특성:**
- 긴 집중 시간 (15초+)
- 추억/향수 선호
- 가족 콘텐츠 관심
- 오후 시간 활용

**관심사:**
- 건강 관리
- 추억, 옛날 이야기
- 손주, 가족
- 여유로운 취미

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 천천히 |
| 톤 | 여유롭고 따뜻한 |
| 훅 | 향수 + 공감 |
| 길이 | 40-55초 권장 |

**예시 문구:**
- "기억하시나요"
- "그때 그 시절"
- "참 좋은 시간이었죠"

---

### 70대 (70-79세)

**미디어 소비 특성:**
- 매우 긴 집중 시간
- 간단한 정보 선호
- 반복 시청 많음
- 낮 시간 활용

**관심사:**
- 건강 유지
- 추억, 역사
- 손주, 가족
- 소소한 일상

**콘텐츠 전략:**
| 항목 | 전략 |
|------|------|
| 속도 | 매우 천천히 |
| 톤 | 친근하고 정서적 |
| 훅 | 따뜻한 감정 |
| 길이 | 35-50초 권장 |

**예시 문구:**
- "참 신기하죠"
- "옛날에는 말이에요"
- "건강하게 지내세요"

## 세대 공통 관심사

### 모든 세대
- 건강
- 돈
- 관계
- 신기한 사실

| 주제 | 적용 세대 |
|------|----------|

| 재테크 | 20-50대 |
| 추억/역사 | 50-70대 |
| 트렌드 | 10-30대 |
