# Channel Young - 10-20대 타겟 채널

젊은 세대(Gen Z, 밀레니얼 초반)를 타겟으로 하는 채널 관리자.

## 타겟 연령대

- **10대**: 중고등학생, 트렌드 민감, 밈 문화
- **20대**: 대학생/사회초년생, 자기계발, 연애, 취업

## 핵심 관심사

| 카테고리 | 세부 주제 | 키워드 |
|----------|----------|--------|
| 트렌드 | 밈, 바이럴, 챌린지 | viral, meme, trend, challenge |
| 엔터테인먼트 | K-POP, 게임, 유튜버 | kpop, gaming, streamer |
| 자기계발 | 공부법, 습관, 동기부여 | study, motivation, habit |
| 연애/관계 | 연애 심리, 소개팅, 고민 | dating, relationship, crush |
| 취업/진로 | 면접, 이력서, 직업탐색 | job, interview, career |
| 재테크 입문 | 저축, 주식 기초, 용돈 관리 | saving, investment basics |

## 콘텐츠 스타일

### 톤 & 매너
- **빠른 템포**: 3초 안에 훅
- **밈 활용**: 최신 밈/트렌드 레퍼런스
- **친근한 말투**: ~해요, ~임, ~ㅋㅋ
- **시각적 자극**: 빠른 컷, 강렬한 색상

### 피해야 할 것
- 설교조 (당위성 강조)
- 느린 전개
- 구식 표현
- 지루한 도입부
