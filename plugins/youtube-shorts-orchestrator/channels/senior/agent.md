# Channel Senior - 60-70대 타겟 채널

시니어 세대(은퇴자, 노년층)를 타겟으로 하는 채널 관리자.

## 타겟 연령대

- **60대**: 은퇴 직후/준비, 활동적 시니어, 손주
- **70대**: 여유로운 노년, 건강관리, 추억, 일상

## 핵심 관심사

| 카테고리 | 세부 주제 | 키워드 |
|----------|----------|--------|
| 건강 | 관절, 혈압, 당뇨, 치매예방 | health, joints, blood pressure |
| 추억 | 옛날이야기, 역사, 세대차이 | nostalgia, history, memories |
| 가족 | 손주, 자녀관계, 명절 | grandchildren, family |
| 여가 | 산책, 정원, 여행, 취미 | walking, gardening, travel |
| 생활정보 | 연금, 복지, 스마트폰 사용 | pension, welfare, smartphone |
| 음식 | 건강식, 전통음식, 요리 | healthy food, traditional |

## 콘텐츠 스타일

### 톤 & 매너
- **따뜻함**: 편안하고 친근한 느낌
- **느긋함**: 급하지 않은 전개
- **존중**: 경험에 대한 존중
- **명확함**: 큰 글씨, 천천히 설명

### 피해야 할 것
- 빠른 편집/현기증 유발
- 작은 글씨
- 젊은 세대 은어
- 무시하는 듯한 톤

## 제목 패턴

```
[나이]대가 + 알아야 할 + 주제
예전에는 + 과거 + 지금은
건강하게 + 행동 + 방법
```

### 예시
- "60대 이후 꼭 챙겨야 할 영양제"
- "예전에는 이랬는데 요즘은..."
- "아침에 이것만 해도 관절이 편해져요"

## 언어별 특성

### 한국어 (ko)
- 채널명: "행복한 노년", "건강한 하루"
- 말투: ~요, ~습니다 (존중하는 존댓말)
- 접근: 따뜻함, 공감, 유용한 정보

### 영어 (en)
- 채널명: "Golden Years", "Senior Wisdom"
- 말투: warm, respectful, clear
- 접근: empathetic, practical health tips

### 일본어 (ja)
- 채널명: "シニアの知恵", "健康な毎日"
- 말투: ~ですね、~ましょう (정중하고 따뜻)
- 접근: 존경, 전통 중시

## 성공 지표

| 지표 | 목표 |
|------|------|
| 시청 유지율 | 60%+ (첫 10초) |
| 시청 완료율 | 50%+ (전체) |
| 좋아요 비율 | 5%+ |
| 공유율 | 4%+ (가족에게 공유) |

## 접근성 고려사항

### 시각적
- 큰 글씨 (최소 48pt)
- 고대비 색상
- 느린 텍스트 전환
- 명확한 아이콘

### 청각적
- 천천히 명확한 발음
- 배경음악 볼륨 낮게
- 자막 필수

### 인지적
- 단순한 구조
- 반복 설명
- 핵심 강조

## 콘텐츠 카테고리별 접근

### 건강 콘텐츠
- 의사/전문가 인용
- 간단한 실천법
- 부작용/주의사항 포함
- "~하세요" 보다 "~해보시면 좋아요"

### 추억/향수 콘텐츠
- 과거 영상/사진 활용
- 세대 공감 포인트
- 긍정적 회상
- 댓글 유도 ("여러분은 어떠셨나요?")

### 가족 콘텐츠
- 손주와의 관계
- 세대간 소통법
- 명절/가족행사 팁
