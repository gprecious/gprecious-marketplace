# gprecious-marketplace

Claude Code 플러그인 마켓플레이스

## 사용 가능한 플러그인

| 플러그인 | 설명 | 리포지토리 |
|---------|------|-----------|
| ultrawork | Sisyphus/Ultrawork 패턴 병렬 에이전트 오케스트레이션 | [gprecious/ultrawork-plugin](https://github.com/gprecious/ultrawork-plugin) |
| youtube-assistant | 유튜브 영상 제작 도우미 (스크립트 첨삭, 트렌드 분석, 음악 추천) | [gprecious/youtube-assistant](https://github.com/gprecious/youtube-assistant) |
| youtube-shorts-orchestrator | 연령대별(10~70대) YouTube Shorts 채널 통합 관리 | plugins/youtube-shorts-orchestrator |

## 설치 방법

```bash
# 특정 플러그인 설치
claude /install youtube-assistant@gprecious-marketplace
claude /install ultrawork@gprecious-marketplace
claude /install youtube-shorts-orchestrator@gprecious-marketplace
```
